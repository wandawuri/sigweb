Esai Materi Yang Telah Dipelajari di Mata Kuliah Sistem Informasi Geografis Berbasis Web

Mata kuliah Praktikum SIG Berbasis Web membahas mengenai bagaimana data dan informasi geospasial ditampilkan melalui internet, baik pada platform berupa desktop, mobile maupun media lainnya. Mata kuliah ini dimulai dari pembahasan mengenai cara kerja internet dan halaman web, konsep dan cara kerja client dan server pada WebGIS, arsitektur WebGIS serta desain dan evaluasi WebGIS.

Pada minggu pertama mempelajari untuk mengenal cara kerja internet dan istilah-istilah internet, mengenal komponen web: HTML, CSS dan JS, latihan membuat web sederhana dengan HTML dan CSS, dan membaca panduan praktik pembuatan web dengan WSL dan Linux.

Pada minggu kedua mempelajari lebih jauh mengenai HTML dan CSS serta penerapannya pada pembuatan halaman Web. Pada minggu ini juga diperkenalkan mengenai Flexbox sebagai perangkat penyusunan layout yang banyak digunakan saat ini.

Pada minggu ketiga merupakan materi kelanjutan dari minggu sebelumnya mengenai Javascript. Pada minggu ini mempelajari peranan Javascript sebagai bahasa pemrograman klien (Client-side scripting) serta dasar-dasar AJAX dan perkembangan Web Modern.

Pada minggu keempat mempelajari mengenai Boostrap sebagai salah satu framework javascript yang paling banyak digunakan

Pada minggu kelima embahas mengenai salah satu webMap API, yaitu LeafletJS. Melakukan pembuatan halaman web sederhana pada LeafletJS, layouting halaman web dengan Bootstrap, membuat layout peta Leaflet dengan Navbar Bootstrap, membuat sebuah tombol pada navbar yang apabila user mengklik tombol tersebut, maka peta akan zoom ke lokasi user saat ini.

Pada minggu keenam membahas mengenai LeafletJS, yaitu penambahan layer, pengenalan format GeoJSON dan instalasi Plugin. Menambahkan plugin ke dalam leaflet untuk memperkaya fungsi webgis. plugin yang bisa ditambahkan (misalnya Leaflet Geolocation, Geocoding, Leaflet Graphic Scale, dll) dapat dilihat pada http://leafletjs.com/plugins.html. Menggunakan QGIS atau GeoJSON.io untuk mengkonversi file Jogja.shp menjadi GeoJSON, kemudian tambahkan ke kerangka peta yang sudah dibuat. Gunakan panduan: https://leafletjs.com/examples/geojson/. Mengkombinasikan bootstraps dengan leaflet yang dibuat (geojson + min 1 plugin) untuk membuat sebuah halaman web. Membuat halaman web yang serupa dengan menggunakan QGIS2Web.

Pada minggu ketujuh dibahas mengenai bagaimana aplikasi Mergin Maps: QGIS in pocket sebagai perangkat berbasis mobile, QGIS sebagai perangkat desktop serta Mergin sebagai aplikasi GIS berbasis online dapat digunakan sebagai sarana untuk pengumpulan data dari lapangan secara kolaboratif. Pengumpulan data menggunakan kombinasi ketiga perangkat tersebut dapat digunakan dalam kegiatan survei lapangan, misalnya terkait dengan survei pelanggaran tata ruang, pemetaan aset, pemetaan tempat makan, pemetaan ATM maupun monitoring pembangunan. Selain itu, mahasiswa secara berkelompok yang terdiri dari maks 5 orang diminta untuk merancang sebuah kegiatan survei berupa Titik, Garis, Poligon dengan total seluruhnya minimal 30 fitur menggunakan kombinasi ketiga perangkat tersebut. Hasilnya dibuat dalam bentuk laporan dan dikumpulkan terakhir satu bulan dari sekarang.
